# Celebal Technologies - Data Science Internship

Comprehensive repository containing all weekly assignments completed during the Data Science internship at Celebal Technologies.

## 📚 Assignments Overview

### Week 1: Python Fundamentals
- Python, NumPy, Pandas, Linear Algebra, Statistics & Probability
- Foundation building for data science

### Week 2: End-to-End ML Pipeline
- Tesla Deliveries Dataset Analysis
- Feature Engineering, Regression, Time Series Forecasting
- Complete EDA and model development

### Week 3: Customer Intelligence System
- Country Development Classification using ML
- Multiple algorithms: Random Forest, XGBoost, SVM, KNN, Naive Bayes
- Ensemble Methods (Stacking) achieving 97.1% accuracy
- Advanced clustering: KMeans, DBSCAN, Agglomerative
- 3D PCA visualization for pattern discovery

### Week 4: Deep Learning on CIFAR-10
- Image Classification using Neural Networks
- ANN Model: 47.11% accuracy on 10 object categories
- CNN Model: 67.86% accuracy with data augmentation
- Comprehensive comparison of architectures
- Convolutional layers, pooling, dropout optimization
