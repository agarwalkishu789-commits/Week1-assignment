# Celebal Technologies - Data Science Internship

Comprehensive repository containing all weekly assignments from the Data Science internship at Celebal Technologies. Each week focuses on different ML and DL techniques with real-world datasets.

## Assignments

*Week 1: Python Fundamentals*
Python, NumPy, Pandas, Linear Algebra, Statistics basics.

*Week 2: End-to-End ML Pipeline*
Tesla Deliveries dataset. Data exploration, feature engineering, regression, time series forecasting.

*Week 3: Classification Project*
Country data classification. Built 6 models - Logistic Regression, Random Forest, XGBoost, SVM, KNN, Naive Bayes. Best: XGBoost 100% accuracy. Ensemble stacking 97.1%. Also did clustering with KMeans, DBSCAN, Agglomerative. 3D PCA visualization.

*Week 4: Deep Learning*
CIFAR-10 image classification. Built ANN (47% accuracy) and CNN (67% accuracy), CNN performed 43% better. Used data augmentation and dropout.

*Week 5: Text generation using RNN, LSTM, and GRU*
Implemented three sequence models on small text corpus. Vanilla RNN performed best with 96.36% accuracy, followed by GRU at 92.73%, while LSTM achieved 72.73%, this demonstrated that model complexity should watch data size, explored vanishing gradient problem, memory mechanisms in gated architectures, and text generation capabilities of each model

## Technologies Used
Python, Pandas, NumPy, Scikit-learn, XGBoost, TensorFlow/Keras, Matplotlib, Seaborn, RNN, LSTM, GRU
